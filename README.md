 AI Crop Disease and Pests Detection

An AI-based web application that helps farmers detect crop diseases and pest infestations using image processing and provides actionable insights for better farming.

✅ Features

Real-time image-based disease and pest detection

Personalized recommendations using LLMs

Soil and weather data integration

Interactive dashboard for crop health insights

Future-ready with IoT integration scope

🧠 Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Python, Flask

AI/ML: CNN (Convolutional Neural Networks), LLMs (Language Models)

Database: SQLite / MySQL

🚀 How to Run

Clone this repository

git clone https://github.com/vushakolaPhanindra/ai-crop-doctor.git
cd ai-crop-doctor

Create and activate a virtual environment

python -m venv venv
source venv/bin/activate  # Linux/Mac
.\venv\Scripts\activate  # Windows

Install dependencies

pip install -r requirements.txt

Start the application

npm start 

📊 Benefits

Helps farmers identify issues early

Reduces dependency on manual diagnosis

Improves crop productivity and sustainability

🌍 Business Scope

Can be extended into a mobile app

Scalable with IoT sensors and cloud hosting

Useful in precision agriculture and AgriTech startups
